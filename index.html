About

  • Create temporary and numbered voice channels by joining a CREATE-Channel

  • Create an associated text channel for each temporary voice channel

  • Easy configurable by using the built in commands. (Type '!dyn help' for an overview)

DynDCM is a free bot for your discord server that dynamically manages channels and automatically creates associated temporary text channels for voice channels.
DynDCM offers an easy to use menu to create, update, delete managed voice channels. DynDCM manages channels dynamically and keep your server tidy and neat. :) You don't need many single channels, which make the server confusing and full.

Just create so-called CREATE-Channels which will automatically create a numbered temporary channel when joining; you will of course also be moved automatically.
Additionally, for each automatically created voice channel an associated temporary text channel is created, which is only visible for the members which are currently in the voice channel.
